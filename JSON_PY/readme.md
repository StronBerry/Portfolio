# JSON Utilities (Python)

Репозиторий, демонстрирующий навыки работы с **JSON-данными в Python**: сериализация, десериализация, преобразование типов и обработка структурированных данных из разных источников:

- портфолио-пример базовой работы с данными,
- набор заготовок для дальнейших проектов,
- демонстрация понимания форматов данных и файлового ввода/вывода.

---

## Ключевые навыки

- работа с модулем `json` в Python;
- сериализация Python-структур (dict, list и др.) в JSON;
- десериализация JSON в Python-объекты;
- обработка вложенных структур данных;
- работа с файлами (`.json`, `.csv`, `.txt`);
- подготовка данных к экспорту и отчётности.

---

## Структура репозитория

### Основные скрипты

- `py_to_json.py`  
  Преобразование Python-структур в JSON-файл.

- `json_to_py.py`  
  Чтение JSON-файлов и преобразование данных в Python-объекты.

- `py_to_any_type.py`  
  Подготовка и преобразование данных различных типов перед сохранением.
